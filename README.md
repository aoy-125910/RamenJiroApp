# 二郎遠征ノート

ラーメン二郎の店舗一覧、訪問記録、行きたい店舗、個人ランキングをスマートフォン中心で管理するためのシングルページアプリです。

## できること

- 店舗一覧の表示
- 訪問済み / 行きたい / 未訪問の管理
- 初訪問日 / 最終訪問日 / メモ / 個人順位の保存
- 達成率とランキングの表示
- 端末内の `localStorage` への保存

## 起動方法

```bash
npm install
npm run dev
```

スマートフォンで確認したい場合は、同じネットワーク上から Vite のローカルURLへアクセスしてください。

## メモ

- 今の店舗データは雛形用の初期セットです
- 将来的に全店舗版へ広げる場合は `src/data/stores.js` を拡張していく想定です
